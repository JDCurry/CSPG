# Requirements

Install required Python packages:

```bash
pip install numpy matplotlib scipy
```
## Usage Notes

1. **All files must be in the same directory** for imports to work correctly
2. **Run scripts from that directory** using `python script_name.py`
3. **Output files appear immediately** in the same directory
4. You can modify output filenames in the scripts if desired
